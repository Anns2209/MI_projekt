Dobrodošli v MI Projekt! To je projekt, ki kombinira Python in HTML tehnologije.

MI Projekt je namenjen raziskavi in razvoju aplikacij s pomočjo Pythona in HTML-ja.

- **Python**: 66.5% - Glavna programska logika
